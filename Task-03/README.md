# 🌳 PRODIGY INFOTECH - Task-03

## 📌 Decision Tree Classifier using Bank Marketing Dataset

### 📖 Objective

Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data using the Bank Marketing dataset from the UCI Machine Learning Repository.

---

## 📂 Dataset

**Dataset Name:** Bank Marketing Dataset

**Source:** UCI Machine Learning Repository

**File Used:** `bank.csv`

The dataset contains information about customers such as:

- Age
- Job
- Marital Status
- Education
- Default Credit
- Housing Loan
- Personal Loan
- Contact Type
- Month
- Duration
- Campaign
- Previous Marketing Outcome
- Target Variable (y)

The target variable indicates whether the customer subscribed to a term deposit.

- **yes** → Customer subscribed
- **no** → Customer did not subscribe
