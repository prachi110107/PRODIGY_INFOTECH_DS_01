# 🚢 Task-02: Data Cleaning and Exploratory Data Analysis (EDA)

## 📌 Objective
Perform Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic dataset to explore relationships between variables and identify meaningful patterns and trends.

## 📂 Dataset
- Dataset: Titanic Dataset
- Source: Kaggle
- File Used: train.csv

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Exploratory Data Analysis Performed

### Data Cleaning
- Removed duplicate records
- Handled missing values
- Checked data types
- Generated summary statistics

### Visualizations
- Survival Count
- Passenger Class Distribution
- Gender Distribution
- Age Distribution
- Survival by Gender
- Survival by Passenger Class
- Correlation Heatmap
- Fare Distribution
- Box Plot (Age vs Survival)

## 📈 Key Insights
- Female passengers had a much higher survival rate than males.
- First-class passengers survived more frequently than passengers in lower classes.
- Younger passengers showed slightly better survival rates.
- Ticket fare had a positive relationship with survival.
- Missing values were mainly present in the Age and Cabin columns.

## 🎯 Learning Outcomes
- Data Cleaning using Pandas
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Data Visualization using Matplotlib and Seaborn
- Extracting Insights from Real-world Data
